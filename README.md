# CSS-Assignment-01-CSS-Selectors
CSS Selectors Assignment
🎯 Goal

practice:

• Element Selector

• Class Selector

• ID Selector

• Group Selector

• Descendant Selector (Combination)



⸻



Step 1: Create index.html



<!DOCTYPE html>

<html>

<head>

 <title>Selectors Practice</title>

 <link rel="stylesheet" href="style.css">

</head>

<body>



 <h1 id="main-heading">My Web Page</h1>



 <p class="intro">This is my intro paragraph.</p>



 <div class="box">

  <h2 class="title">Box Title</h2>

  <p>This paragraph is inside the box.</p>

  <a href="#">Click Here</a>

 </div>



 <div class="box">

  <h2 class="title">Another Box</h2>

  <p>This is another paragraph inside box.</p>

  <a href="#">Learn More</a>

 </div>



</body>

</html>





⸻



Step 2: Create style.css



1) Element Selector



Make all p color gray.



⸻



2) ID Selector



Make #main-heading color blue.



⸻



3) Class Selector



Make .intro color green.



⸻



4) Class Selector



Make .box text color brown.



⸻



5) Group Selector



Make h1 and h2 color purple.



⸻



6) Descendant Selector (Combination Selector)



Make only the a inside .box color red.
